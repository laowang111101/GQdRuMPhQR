# 项目简介 python573

本项目是一款基于Python语言的宠物在线商城系统，采用Django框架进行开发，使用MySQL数据库进行数据存储。

## 技术栈

- 后端：Python 3.x，Django
- 数据库：MySQL
- 前端：HTML，CSS，JavaScript

## 功能模块

1. 用户模块
   - 用户注册与登录
   - 用户信息管理
   - 用户购物车管理
2. 商品模块
   - 商品展示
   - 商品搜索
   - 商品分类
3. 订单模块
   - 下单与支付
   - 订单状态查询
   - 订单管理

## 系统角色

- 普通用户：浏览商品，购买商品，管理个人信息
- 管理员：管理商品，处理订单，监控系统运行状态

## 运行环境

- 操作系统：Linux，Windows
- Python版本：3.x
- 依赖库：Django，MySQL Connector/Python

## 部署步骤

1. 安装Python 3.x
2. 安装MySQL数据库
3. 克隆项目代码
4. 安装依赖库
5. 配置数据库连接
6. 运行项目
